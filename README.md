# AI Powered Debt Relief & Financial Recovery Platform

## Project Overview

The AI Powered Debt Relief & Financial Recovery Platform is an intelligent web application that assists borrowers in managing debt through financial analysis, AI-powered settlement recommendations, and automated negotiation letter generation.

## Technologies Used

- React.js + Vite
- FastAPI
- Python
- SQLite
- SQLAlchemy
- Google Gemini AI
- JWT Authentication
